GRAPHIC DESIGN

man-made vertigo Lyric Book

❮ ❯

Click through the slideshow above to view each page of lyrics.

To go along with my album man-made vertigo, I created a zine-style lyric book to practice my typography and graphic design skills in while visually representing each song. This was my first time using Adobe Illustrator, which I taught myself throughout the process. I paid close attention to how each track feels sonically and lyrically, aiming to translate that emotion into a visual language that felt true to the music.
